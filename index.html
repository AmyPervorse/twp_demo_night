Projects I've Created

I created this website while folowing along with the instructor during the Tech Workforce Program. The website contains unordered lists, an imbeded map, CSS styling, loops, and even a Javascript game! I used Atom to edit the code, Github to push to the repository, and Firebase to deploy to the internet.

I created this simple website in order to be considered as an appilcant for the Tech Workforce Program. I took an HTML course in high school but was unfamiliar with CSS. After taking CodeAcademy's CSS course, I was able to incorporate what I'd learned into my website.
